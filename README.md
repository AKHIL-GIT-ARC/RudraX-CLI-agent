# ⚡ RudraX — CLI AI Assistant

A powerful voice-enabled command-line AI assistant built with Python.

## Features
- 🗣️ Voice input
- 🧠 AI chat (Claude API)
- 🖼️ Solve problems from images
- 🌤️ Live weather
- 🔍 Google search
- 🕐 Time and date
- 🚀 Open Linux apps

## Setup
1. Clone: git clone https://github.com/AKHIL-GIT-ARC/RudraX-CLI-agent
2. Activate: source venv/bin/activate
3. Install: pip install -r requirements.txt
4. Add keys: cp .env.example .env then edit .env
5. Run: python main.py

## Commands
| What you type | What happens |
|---|---|
| what time is it | Shows current time |
| open firefox | Opens Firefox |
| search python tips | Google search |
| weather in Mumbai | Live weather |
| solve 2x + 5 = 15 | AI solves it |
| who made the internet | AI answers |
